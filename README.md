## Requirements

- PHP 8.0 or later
- Nodejs 16.17 or later


## Installing Project
```bash
composer install
```
```bash
npm install
```

## Running Project
Open two terminal tabs and from each tab run the below commands:

```bash
npm run dev
```

```bash
php artisan serve
```
